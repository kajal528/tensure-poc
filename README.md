# Tensure Landing Page

A recreation of the [Tensure landing page](https://tensure-landing.lovable.app/) built with React, Vite, TypeScript, Tailwind CSS v4, and Framer Motion.

**Live site:** [https://kajal528.github.io/tensure-poc/](https://kajal528.github.io/tensure-poc/)

## Stack

- React 19 + Vite
- TypeScript
- Tailwind CSS v4
- Framer Motion

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

## Deploy

Pushes to `main` automatically deploy to GitHub Pages via GitHub Actions.

Enable **Settings → Pages → Build and deployment → GitHub Actions** on the repository if the site does not appear after the first push.
